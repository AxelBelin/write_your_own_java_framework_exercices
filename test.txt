Q5 :
Que se passe-t-il ? Pourquoi ? Comment corriger le problème ? 
Pour que les modifs soient bien prises en compte sur le repo distant, il faut utiliser les commandes suivantes :
git add <fichiers>
git commit -m "message de commit"
git push


Q6 :
Comment faire pour mettre à jour la copie local de votre repository pour que vous voyiez le contenu du fichier changé sur la machine locale.
Il faut faire un git pull.
En cas de conflit, on peut faire un merge ou un rebase.
